# 🚚 Freight Cost Prediction System

## 📌 Project Description
This project predicts freight cost based on Distance, Weight, and Fuel Price using Machine Learning.

## ⚙️ Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- HTML, CSS

## 🧠 Machine Learning Model
- Random Forest Regressor
- Linear Regression (for comparison)
- Evaluation Metrics: MAE, R2 Score

## 📊 Features
- User input through web interface
- Real-time cost prediction
- Graphical analysis
- Clean and attractive UI

## 🚀 How to Run

```bash
git clone https://github.com/sindhu834/Freight-Cost-Prediction.git
cd Freight-Cost-Prediction
pip install -r requirements.txt
python app.py
